# 📸 3D Object Placement & Photography System
**(3Dオブジェクト配置・撮影システム)**

ブラウザ上で手軽に3Dオブジェクト（人物・建物・外部GLBモデル）を配置し、ライティングや背景を調整して、好きな構図で写真（PNG画像）を撮影できるWebアプリケーションです。

## 主な機能 (Features)

###  配置モード (Placement Mode)
* **プリセットモデル**: 関節（腕・脚・向き）を動かせる「人物」と、サイズや色を変えられる「建物」をワンクリックで追加可能。
* **外部モデルの読み込み**: お手持ちの `.glb` または `.gltf` ファイルを画面にドラッグ＆ドロップするだけで即座に配置。
* **直感的なギズモ操作**: オブジェクトの移動・回転・拡大縮小をマウスで直感的に操作できます。
* **便利な編集機能**: Ctrl+C / Ctrl+Vによる複製や、Ctrl+Zによる操作のやり直し（Undo機能）に対応。
* **アウトライナー**: 画面右下のオブジェクト一覧リストから、重なったモデルも簡単に選択可能。

###  撮影モード (Camera Mode)
* **アスペクト比設定**: 用途に合わせて撮影サイズ（16:9, 9:16, 1:1）を切り替え可能。
* **精密なカメラ制御**: スライダーを使って、カメラのX/Y/Z座標、注視点（パン）、画角（ズーム）をミリ単位で調整。マウス操作とも完全同期します。
* **ワンクリックPNG保存**: 「撮影」ボタンを押すだけで、現在のキャンバス状態をきれいなPNG画像としてダウンロード。

###  環境・ライティング設定 (Environment & Lighting)
* **360度パノラマ背景**: 正距円筒図法（Equirectangular）のパノラマ画像（JPG/PNG）をドロップすると、全天球背景として展開され、モデルへの環境光（映り込み）としても適用されます。
* **照明のカスタマイズ**: 全体の明るさや、太陽光（DirectionalLight）の強さ・色・角度を自由に調整し、夕景や夜のシーンも作成可能。
* **地面・グリッドの表示切替**: 撮影の邪魔になる場合はワンクリックで非表示にできます。

###  プロジェクト管理 (Save & Load)
* **セーブ＆ロード**: 現在の配置、カメラ位置、環境設定を `.json` ファイルとして書き出し・読み込み可能。
* **アセットのバンドル**: 読み込んだGLBモデルやパノラマ画像もBase64エンコードされてJSONに保存されるため、ファイル1つで完全に状態を復元できます。

## 使い方 (Usage)

### セットアップ
特別なビルド環境は不要です。`project
.html` をブラウザ（Chrome, Edge, Firefox, Safari等）で開くだけで動作します。
（※ ローカルサーバー環境での起動を推奨します。VS Code拡張機能の「Live Server」などが便利です。）

### 基本操作ショートカット
配置モード時に、オブジェクトを選択した状態で以下のキーボード操作が可能です。

| キー | アクション |
| --- | --- |
| `T` | 移動モード (Translate) |
| `R` | 回転モード (Rotate) |
| `S` | 拡大縮小モード (Scale) |
| `Delete` / `Backspace` | 選択オブジェクトの削除 |
| `Ctrl + C` | 選択オブジェクトのコピー |
| `Ctrl + V` | オブジェクトのペースト (複製) |
| `Ctrl + Z` | 操作のやり直し (Undo) |
